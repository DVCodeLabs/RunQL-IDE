## <a id="reh"></a>What are reh and reh-web archives?

- Remote Host (`reh`) is the server-side component used for remote scenarios such as SSH or WSL.
- Web Host (`reh-web`) is the server component behind `serve-web` style browser access to the editor.

These archives are part of the upstream Code OSS and VSCodium build layout and are relevant mainly when you package or distribute remote and browser-hosted variants of RunQL.
